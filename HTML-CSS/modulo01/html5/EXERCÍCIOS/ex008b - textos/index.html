def calcular_soma ():
    numero= 0
    for i in range(1,11):
        numero+= i
return numero

resultado = calcular_soma()
print("A soma dos números de 1a 10 é:") resultado)
os atalhos utilizados para tirar o texto do "centro" e ajustar ao máximo no lado esquerdo foram SHIT + TAB se por acaso quiser voltar o texto para o lugar de antes use somente a tecla TAB

Explicação:

  1. A linha numero = 0 foi indentada com 4 espaços para estar dentro da função calcular_soma().
  2. A linha numero += i foi indentada com mais 4 espaços para estar dentro do bloco for.
  3. A linha return numero foi indentada com 4 espaços para estar dentro da função calcular_soma().
